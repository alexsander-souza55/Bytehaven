# BYTEHAVEN — THE LOST CONTAINERS

## Sobre
Jogo 2D platformer desenvolvido em Python com Pygame.
Atividade Prática — Linguagem de Programação Aplicada — UNINTER 2026.

## Como jogar
- **A / D** — Mover esquerda / direita
- **W / SPACE** — Pular
- **CTRL** — Atirar
- Colete todos os Containers de cada fase para avançar.
- Não caia em zonas corrompidas!

## Requisitos
```
pip install -r requirements.txt
```

## Executar
```
python main.py
```

## Stack
- Python 3.12
- Pygame 2.6.0
- SQLite3
